## 2、img 标签

简单介绍：

图像标签，是image的缩写。

### 语法：

```
<img src="图像URL" alt=""/>
```

### 主要属性：

|  属性  |  值  |               描述               |
| :----: | :--: | :------------------------------: |
|  src   | URL  |            图像的路径            |
| title  | 文本 | 图像不能显示时的替换图片内容介绍 |
|  alt   | 文本 |       鼠标悬停时显示的内容       |
| width  | 像素 |             图片宽度             |
| height | 像素 |             图片高度             |
